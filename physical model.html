Physical model atau model fisik dalam basis data adalah level terendah dari abstraksi data yang berfokus pada detail penyimpanan fisik data di perangkat keras. Model ini menggambarkan bagaimana data benar-benar disimpan dalam sistem penyimpanan, termasuk struktur tabel, nama dan nilai kolom, kunci utama dan asing, hubungan antar tabel, serta teknik pengindeksan dan organisasi file yang digunakan untuk mengoptimalkan kinerja basis data. Model fisik biasanya dibuat menggunakan bahasa spesifik sistem manajemen basis data (DBMS) dan bertujuan untuk meningkatkan efisiensi penyimpanan serta akses data.
